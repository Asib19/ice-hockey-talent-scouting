## Project Overview
An exploratory data mining and modeling pipeline analyzing hockey player performance data to support talent scouting decisions. The project covers data cleaning, correlation analysis, outlier detection, and predictive modeling across multiple player attributes (performance stats, physical measurements, contract data, and injury history).

## Structure
- `1_data_cleaning.ipynb` — data cleaning and preprocessing
- `2_modeling.ipynb` — modeling and analysis
- `correlation/` / `graphs/` — correlation, outlier, and exploratory visualizations
- `orange/` — Orange Data Mining workflow (`Orange.ows`) used for visual pipeline exploration
- `data/` — raw and processed datasets
